    Q1:

    近义词

    • opponent = rival 对手
    • disintegrating = breakdown = collapse 分解 瓦解

    • contradictions = conflict 矛盾

    • shattered = destroyed
    • oblige = responsibility 责任
    • degenerated = degrade 降低
    • drudgery = dull work 枯燥的工作
